Developing product project

This is developing data course project has two parts. First, to create a Shiny application and deploy it on Rstudio's servers. Second, use Slidify or Rstudio Presenter to prepare a reproducible pitch presentation about the application.

Introduction

we are usind Diamond Explorer project which shows the relation between prices and 54,000 different type of diamonds.

1- you can select small sample or the whole data.

2- you can select different columns to compare with in X and Y axis.

3- can select to smooth or jitter graph type to be added to the diagram.

4- in addition, you can diffrentiate the diagram to different group by selecting color facet. Ex: group by cuts or prices
